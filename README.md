# Splunk-SOC-Investigation-Powershell
## Project Overview

This project simulates a Tier 1 SOC investigation using Splunk. The investigation focuses on a high-severity alert involving suspicious login activity, encoded PowerShell execution, and outbound network connections from a Windows workstation.

The goal of this project was to ingest log data into Splunk, investigate suspicious activity using SPL searches, reconstruct the incident timeline, identify indicators of compromise, map the activity to MITRE ATT&CK, build a SOC dashboard, and produce a final analyst report.
